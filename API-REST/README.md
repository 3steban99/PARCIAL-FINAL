# MAXI KIOSCO

Hola¡ este es una **API REST** creada con **express** del crud del kiosco (creado en react) para el parcial final de la materia **Programacion 2** de la carrera Tecnico Superior en Programacion. Esta api cuenta con las llamadas para el crud: llamado get, delete, push y put. La base de datos esta creada en MySQL mendiante el XAMPP. La base cuenta con las tablas necesarias para el crud del kiosco: Productos, Forma de Pago, Facturas, Detalle Factura.

# INSTALACION

 1. Clona el repositorio en tu máquina **local**:
	 `git clone https://github.com/3steban99/PARCIAL-FINAL.git`

	`cd PARCIAL-FINAL API-REST`
	
	`npm install`
	
	`npm start`
